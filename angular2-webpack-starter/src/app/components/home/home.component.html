Project description
This is my BSc level professional dissertation in Budapest University of Technology and Economic. The purpose of the project is to make a web based application which gratifies test compliting to the users. Users have to be able to make Categories like subjects then make test scenarios what other users can fill out in frame of exams.
Unfortunately my colleague had given up the project and his job was the server implementation. To elinimate and simulate the full progress of the application I've created a fake server on the client side what represents the Angular2 In-memory web application. This fake server knows only the getter and setter functions of most functionatelly so it may happen some functions aren't working. I apologize for that.
Now there are four main roles what you can choose:
  • Administrator
  • Manager
  • User
  • Developer
Whose have different access for the modules. To check all possibilities here are their username-password combinations:
  • admin/admin -> Administrator role
  • manager/manager -> Manager role
  • user/user -> User role
  • mbalyi/test -> Developer role
